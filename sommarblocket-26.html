GitHub-synk

Loggarna sparas automatiskt i ett privat repo på ditt GitHub-konto, och hämtas tillbaka när du öppnar appen — på vilken enhet som helst.

Inte ansluten.
Så skapar du repo + nyckel (engångsgrej, 3 min)
  1. På github.com: skapa ett nytt privat repo, t.ex. traningslogg. Bocka i "Add a README" så repot inte är tomt.
  2. Gå till Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token.
  3. Repository access: Only select repositories → välj bara traningslogg.
  4. Permissions → Repository permissions → Contents: Read and write. Inget annat.
  5. Sätt gärna lång giltighetstid. Skapa, kopiera nyckeln och klistra in ovan.

Nyckeln sparas bara i webbläsaren på den här enheten — den laddas aldrig upp någonstans utom till GitHubs eget API. Dela den aldrig med någon. På en ny enhet klistrar du in samma uppgifter en gång.
